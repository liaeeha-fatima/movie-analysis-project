# 📊 Customer Churn Analysis & Data Cleaning

This repository contains a professional data analysis workflow focused on **Customer Churn**. The project demonstrates the ability to handle raw business data, perform cleaning using Python, and prepare datasets for predictive modeling.

## 🎯 Project Objective
The goal is to analyze customer behavior and identify patterns that lead to churn. By merging customer demographics with order history, we create a unified view of the customer journey.

## 🛠️ Technical Stack
* **Language:** Python 3.x
* **Libraries:** Pandas (Data Wrangling), NumPy
* **Tools:** GitHub for Version Control

## 📂 Repository Structure
* `main.py`: The core Python script that automates the data cleaning process.
* `customers.csv`: Dataset containing customer profiles and registration details.
* `orders.csv`: Transactional data including order dates and amounts.
* `LICENSE`: MIT License (Open Source).

## ⚙️ How it Works
The `main.py` script performs the following:
1. Loads raw CSV datasets.
2. Handles missing values and data type conversions.
3. Prepares the data for Exploratory Data Analysis (EDA).
